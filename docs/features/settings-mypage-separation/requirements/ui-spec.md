## 2. /mypage 페이지

### 레이아웃

```
┌─────────────────────────────────────────┐
│  ← 뒤로가기        마이페이지            │
├─────────────────────────────────────────┤
│                                         │
│        ┌───────────┐                    │
│        │   🧑‍💻    │  ← 프로필 이미지    │
│        │  [편집]   │                    │
│        └───────────┘                    │
│         홍길동                          │
│         user@email.com                  │
│         ✅ 이메일 인증됨                 │
│                                         │
├─────────────────────────────────────────┤
│                                         │
│  ┌─────────────────────────────────┐   │
│  │ 프로필 관리                      │   │
│  ├─────────────────────────────────┤   │
│  │ 프로필 수정                   → │   │
│  │ 비밀번호 변경                 → │   │
│  │ 알림 설정                     → │   │
│  └─────────────────────────────────┘   │
│                                         │
│  ┌─────────────────────────────────┐   │
│  │ 내 활동                         │   │
│  ├─────────────────────────────────┤   │
│  │ 내 게시글                     → │   │
│  │ 내 댓글                       → │   │
│  │ 좋아요한 글                   → │   │
│  │ 북마크                        → │   │
│  └─────────────────────────────────┘   │
│                                         │
│  ┌─────────────────────────────────┐   │
│  │ 계정                            │   │
│  ├─────────────────────────────────┤   │
│  │ [로그아웃]                       │   │
│  │                                 │   │
│  │ 계정 탈퇴                     → │   │
│  └─────────────────────────────────┘   │
│                                         │
└─────────────────────────────────────────┘
```

### 서브 페이지

| 경로 | 설명 |
|------|------|
| `/mypage` | 메인 (위 레이아웃) |
| `/mypage/edit` | 프로필 수정 폼 |
| `/mypage/password` | 비밀번호 변경 폼 |
| `/mypage/notifications` | 알림 설정 |
| `/mypage/posts` | 내 게시글 목록 |
| `/mypage/comments` | 내 댓글 목록 |
| `/mypage/likes` | 좋아요한 글 목록 |
| `/mypage/bookmarks` | 북마크 목록 |

### 컴포넌트 명세

| 컴포넌트 | 타입 | Props |
|----------|------|-------|
| ProfileHeader | Display | user |
| ProfileEditForm | Form | user, onSubmit |
| PasswordChangeForm | Form | onSubmit |
| ActivityList | List | type, items |
| LogoutButton | Button | onLogout |

### 상호작용

- 프로필 이미지 클릭: 이미지 업로드 모달
- 로그아웃: 확인 다이얼로그 후 실행
- 계정 탈퇴: 비밀번호 재입력 + 확인 다이얼로그
