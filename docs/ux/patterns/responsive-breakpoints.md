# 반응형 브레이크포인트 전략

## UX-MVP-006: 반응형 디자인 가이드라인

### 개요
다양한 디바이스에서 최적의 사용자 경험을 제공하기 위한 반응형 브레이크포인트 전략을 정의합니다.

---

## 1. 브레이크포인트 정의

### 1.1 주요 브레이크포인트

| 이름 | 범위 | 주요 디바이스 |
|------|------|--------------|
| `xs` (Extra Small) | 0 - 479px | 소형 스마트폰 |
| `sm` (Small) | 480 - 767px | 스마트폰 |
| `md` (Medium) | 768 - 1023px | 태블릿 |
| `lg` (Large) | 1024 - 1279px | 소형 데스크톱, 태블릿 가로 |
| `xl` (Extra Large) | 1280 - 1535px | 데스크톱 |
| `2xl` (2X Large) | 1536px+ | 대형 데스크톱 |

### 1.2 CSS 변수 정의

```css
:root {
  --breakpoint-xs: 0px;
  --breakpoint-sm: 480px;
  --breakpoint-md: 768px;
  --breakpoint-lg: 1024px;
  --breakpoint-xl: 1280px;
  --breakpoint-2xl: 1536px;
}
```

### 1.3 Tailwind CSS 설정

```javascript
// tailwind.config.js
module.exports = {
  theme: {
    screens: {
      'xs': '480px',
      'sm': '640px',
      'md': '768px',
      'lg': '1024px',
      'xl': '1280px',
      '2xl': '1536px',
    },
  },
}
```

## 4. 타이포그래피 반응형

### 4.1 폰트 크기 스케일

| 요소 | xs-sm | md | lg+ |
|------|-------|-----|------|
| h1 | 24px | 28px | 32px |
| h2 | 20px | 22px | 24px |
| h3 | 18px | 18px | 20px |
| body | 14px | 15px | 16px |
| small | 12px | 12px | 14px |
| caption | 11px | 12px | 12px |

### 4.2 CSS 구현

```css
/* 모바일 우선 접근 */
:root {
  --font-size-h1: 24px;
  --font-size-body: 14px;
  --line-height-body: 1.5;
}

@media (min-width: 768px) {
  :root {
    --font-size-h1: 28px;
    --font-size-body: 15px;
  }
}

@media (min-width: 1024px) {
  :root {
    --font-size-h1: 32px;
    --font-size-body: 16px;
  }
}

/* clamp() 활용 */
h1 {
  font-size: clamp(24px, 4vw, 32px);
}
```

## 9. 스크롤 동작

### 9.1 헤더 스크롤 반응

```
모바일 (sm-):
- 아래 스크롤: 헤더 숨김 (48px 이상 스크롤 시)
- 위 스크롤: 헤더 표시
- 맨 위 도달: 헤더 고정 표시

태블릿/데스크톱 (md+):
- 헤더 항상 표시 (고정)
```

### 9.2 사이드바 스크롤

```
데스크톱:
- 좌측 사이드바: sticky, 뷰포트 높이 내에서 고정
- 우측 사이드바: sticky, 콘텐츠와 함께 스크롤 후 고정
```

## 10. 성능 최적화

### 10.1 조건부 로딩

```javascript
// 데스크톱 전용 컴포넌트
const SidebarRight = dynamic(
  () => import('./SidebarRight'),
  { ssr: false }
);

// 미디어 쿼리로 조건부 렌더링
const isDesktop = useMediaQuery('(min-width: 1280px)');
{isDesktop && <SidebarRight />}
```

### 10.2 Critical CSS

| 브레이크포인트 | Critical CSS 대상 |
|--------------|------------------|
| 모바일 우선 | 모바일 레이아웃 + 첫 화면 스타일 |
| 지연 로드 | 데스크톱 전용 스타일 |
