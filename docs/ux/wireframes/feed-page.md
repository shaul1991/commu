# 메인 피드 페이지 와이어프레임

## UX-MVP-002: 피드 페이지 레이아웃

### 개요
메인 피드 페이지는 사용자가 커뮤니티의 게시글을 탐색하는 핵심 진입점입니다.

## 컴포넌트 상세 명세

### 1. 헤더 (Header)

| 요소 | 설명 | 동작 |
|------|------|------|
| 로고 | 서비스 로고 | 클릭 시 피드 최상단 이동 |
| 검색바 | 게시글 검색 | 입력 시 실시간 자동완성 |
| 알림 아이콘 | 미확인 알림 표시 | 알림 드롭다운 열기 |
| 프로필 | 사용자 아바타 | 프로필 드롭다운 메뉴 |

### 2. 게시글 카드 (Post Card)

```
┌─────────────────────────────────────────────────┐
│  ┌────┐                                         │
│  │ 📷 │  [작성자명]  •  2시간 전                │
│  │    │  ─────────────────────────────────────  │
│  └────┘  제목: 게시글 제목이 여기에 표시됩니다  │
│          본문 미리보기 텍스트가 최대 2줄까지    │
│          표시되며 그 이상은 ...으로 생략됩니다  │
│          ─────────────────────────────────────  │
│          ❤️ 42    💬 12    🔖 북마크            │
│          좋아요   댓글     저장                 │
└─────────────────────────────────────────────────┘
```

#### 카드 구성 요소

| 요소 | 크기/제한 | 설명 |
|------|----------|------|
| 썸네일 | 80x80px (데스크톱) | 첫 번째 이미지 또는 기본 아이콘 |
| 제목 | 최대 1줄 | 초과 시 말줄임 처리 |
| 미리보기 | 최대 2줄 | 본문 텍스트 미리보기 |
| 작성자 | - | 아바타 + 이름 |
| 메타 정보 | - | 좋아요, 댓글 수, 작성 시간 |

### 3. 사이드 네비게이션

| 메뉴 | 아이콘 | 기능 |
|------|-------|------|
| 홈 | 🏠 | 전체 피드 |
| 인기 | 🔥 | 인기 게시글 |
| 내 글 | 📝 | 본인 작성 글 |
| 북마크 | 🔖 | 저장한 글 |

### 4. 플로팅 액션 버튼 (모바일)

```
      ┌───┐
      │ + │  ← 글쓰기 버튼
      └───┘
         위치: 우측 하단
         크기: 56x56px
         색상: Primary Color
```

## 인터랙션 정의

### 스크롤 동작

| 동작 | 결과 |
|------|------|
| 아래로 스크롤 | 헤더 숨김 (모바일), 새 게시글 로드 |
| 위로 스크롤 | 헤더 표시 |
| 맨 아래 도달 | 무한 스크롤 트리거 |

### 게시글 카드 호버 (데스크톱)

```
기본 상태:                    호버 상태:
┌─────────────────┐          ┌─────────────────┐
│                 │          │  ░░░░░░░░░░░░░  │ ← 배경색 변경
│  게시글 카드    │   ──▶    │  게시글 카드    │
│                 │          │                 │
└─────────────────┘          └─────────────────┘
                              그림자 효과 추가
```

### Pull to Refresh (모바일)

```
    ↓ 당겨서 새로고침
    ┌──────────────┐
    │    ◌        │ ← 로딩 인디케이터
    │    ↻        │
    └──────────────┘
```
