# 게시글 상세 페이지 와이어프레임

## UX-MVP-002: 게시글 상세 페이지 레이아웃

### 개요
게시글 상세 페이지는 사용자가 개별 게시글의 전체 내용을 확인하고 상호작용하는 페이지입니다.

## 컴포넌트 상세 명세

### 1. 게시글 헤더

```
┌─────────────────────────────────────────────────────────────┐
│                                                             │
│  ┌─────┐   작성자명                                         │
│  │ 👤  │   @username  •  2시간 전  •  조회 123              │
│  │     │                                                    │
│  └─────┘   [팔로우]                           [⋮ 더보기]    │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

| 요소 | 설명 |
|------|------|
| 아바타 | 48x48px, 클릭 시 프로필 이동 |
| 작성자명 | 굵은 글씨, 클릭 시 프로필 이동 |
| 작성 시간 | 상대 시간 표시 (예: 2시간 전) |
| 팔로우 버튼 | 본인 글에는 미표시 |
| 더보기 | 수정/삭제/신고 메뉴 (본인 글: 수정/삭제, 타인 글: 신고) |

### 2. 게시글 본문

```
┌─────────────────────────────────────────────────────────────┐
│                                                             │
│  게시글 제목 (h1)                                           │
│  ═══════════════════════════════════════════════════════════│
│                                                             │
│  본문 내용                                                  │
│                                                             │
│  - 마크다운 문법 지원                                       │
│  - 이미지 인라인 표시                                       │
│  - 코드 블록 하이라이팅                                     │
│  - 링크 자동 감지                                           │
│                                                             │
│  ┌────────────────────────────────────────────────────────┐ │
│  │                                                        │ │
│  │                    📷 이미지                           │ │
│  │                                                        │ │
│  │            (클릭 시 라이트박스 열림)                   │ │
│  │                                                        │ │
│  └────────────────────────────────────────────────────────┘ │
│                                                             │
│  #태그1  #태그2  #태그3                                     │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

### 3. 인터랙션 바

```
┌─────────────────────────────────────────────────────────────┐
│                                                             │
│   ❤️         💬         🔖         📤                       │
│   42        12         저장       공유                      │
│  좋아요     댓글       북마크                               │
│                                                             │
│   ↓ 클릭 시                                                 │
│   ♥️ (채워진 하트 + 숫자 증가 애니메이션)                   │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

| 버튼 | 액션 | 피드백 |
|------|------|--------|
| 좋아요 | 좋아요 토글 | 아이콘 변경 + 숫자 애니메이션 |
| 댓글 | 댓글 영역으로 스크롤 | 댓글 입력창 포커스 |
| 북마크 | 북마크 토글 | 아이콘 변경 + 토스트 메시지 |
| 공유 | 공유 모달 열기 | 링크 복사/SNS 공유 옵션 |

## 스크롤 동작

| 스크롤 위치 | 동작 |
|------------|------|
| 상단 | 헤더 표시 |
| 중간 (본문) | 헤더 숨김 (모바일) |
| 하단 (댓글) | 플로팅 댓글 입력창 표시 (모바일) |
| 인터랙션 바 통과 | 플로팅 인터랙션 바 표시 (선택적) |
